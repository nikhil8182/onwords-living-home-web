Home Assistant Setup

Home Assistant Configuration Required

To use this application, you need to provide your Home Assistant details:

Option 1: Using config.env File (Recommended)

Edit the config.env file in the application directory:

# Home Assistant Configuration
HA_URL=http://your-home-assistant-url:8123
HA_TOKEN=your_long_lived_access_token

Option 2: Using Environment Variables

Set the following environment variables:

export HA_URL=http://your-home-assistant-url:8123
export HA_TOKEN=your_long_lived_access_token

How to Create a Long-Lived Access Token

  1. In Home Assistant, click on your user profile from the sidebar
  2. Scroll to the bottom to the "Long-Lived Access Tokens" section
  3. Click "Create Token"
  4. Give it a name like "Flask App" and click "OK"
  5. Copy the token (you won't be able to see it again)
  6. Set the token in the config.env file or as an environment variable

Restart the Application

After setting the configuration, restart this application.
